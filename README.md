# Hanayo(Underdeveloped)
基于node的一个静态博客站点生成器
A static blog generator writed by Node.js
# Usage

## Install

```shell
npm install hanayo -g
```
## Init proj

```shell
hanayo init
```
## Build files

```shell
hanayo build
```

## Start server

```shell
hanayo server
```

then open http://127.0.0.1:8000
